# FastAPI Scaffold with Auth 🚀

A **production-ready FastAPI template** with everything you need to build secure, scalable APIs fast.

## Features ✨

- **🔐 JWT Authentication** - Secure token-based authentication with access & refresh tokens
- **👤 User Management** - Complete user CRUD with email validation
- **📊 Database Migrations** - Alembic integration for version-controlled schema changes
- **⚡ Rate Limiting** - Built-in request rate limiting for API protection
- **📝 Structured Logging** - JSON logging with proper error handling
- **🏥 Health Checks** - Liveness and readiness endpoints
- **🗄️ Database Agnostic** - Support for SQLite, PostgreSQL, and MySQL
- **🏗️ Modular Architecture** - Clean separation of concerns
- **📚 Full Documentation** - API docs auto-generated by FastAPI
- **🧪 Test Ready** - Pre-configured pytest + fixtures

## Quick Start 🎯

### Using This Repository as a GitHub Template

1. Click **"Use this template"** on GitHub
2. Clone your new repository
3. Copy `.env.example` to `.env` and fill in values
4. Install dependencies: `pip install -e ".[dev]"`
5. Run migrations: `alembic upgrade head`
6. Start dev server: `fastapi dev example_project/app/main.py`

### Installing as a Python Package

```bash
pip install fastapi-scaffold-with-auth
```

Then generate a new project:

```bash
python -m fastapi_scaffold_with_auth scaffold my_api
```

## Configuration ⚙️

Create a `.env` file in the project root:

```env
# Security
SECRET_KEY=your-secret-key-at-least-32-characters-long
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
REFRESH_TOKEN_EXPIRE_DAYS=7

# Database
DATABASE_URL=sqlite:///./app.db
# For PostgreSQL: postgresql://user:password@localhost/dbname
# For MySQL: mysql+pymysql://user:password@localhost/dbname

# API
API_V1_STR=/api/v1
DEBUG=False

# Logging
LOG_LEVEL=INFO
LOG_FORMAT=json  # or 'text'
```

## Authentication Flow 🔐

### Register a User

```bash
curl -X POST "http://localhost:8000/api/v1/auth/register" \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"securepass123"}'
```

### Login & Get Tokens

```bash
curl -X POST "http://localhost:8000/api/v1/auth/login" \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"securepass123"}'
```

Response:

```json
{
  "access_token": "eyJhbGc...",
  "refresh_token": "eyJhbGc...",
  "token_type": "bearer"
}
```

### Use Access Token

```bash
curl -H "Authorization: Bearer {access_token}" \
  "http://localhost:8000/api/v1/users/me"
```

## Database Migrations 📊

### Create a New Migration

```bash
alembic revision --autogenerate -m "Add new field to users"
```

### Apply Migrations

```bash
alembic upgrade head
```

### Rollback

```bash
alembic downgrade -1
```

## Rate Limiting 🚦

Rate limits are configured per endpoint. Example:

```python
from app.middleware.rate_limit import limiter
from fastapi import FastAPI

@app.get("/api/endpoint")
@limiter.limit("5/minute")
async def my_endpoint(request: Request):
    return {"message": "ok"}
```
